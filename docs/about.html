About the Project

SIH-2K25 ERP is a full-fledged academic resource planning platform. It connects students, faculty, and administrative workflows including admissions, courses, exams, results, hostel management, fees and payments, library, and attendance.

Tech Stack

  • Backend: ASP.NET Core 8, CoreWCF SOAP, Dapper, PostgreSQL
  • Frontend: React 18, Vite, Tailwind CSS, React Router, Context + useReducer
  • Testing: xUnit
  • Docs: GitHub Pages
