SIH-2K25 ERP Project

A modern, modular Educational Resource Planning system built with ASP.NET Core 8, CoreWCF SOAP, PostgreSQL, and React + Vite frontend.

View on GitHub Get Started

Key Features

Full ERP Modules

Students, Courses, Departments, Users, Fees/Payments, Exams, Library, Hostel, Attendance, and more.

Production-Ready Backend

ASP.NET Core 8 + CoreWCF SOAP, Dapper, PostgreSQL, CORS, health checks, error middleware.

Modern Frontend

React 18 + Vite + Tailwind, React Router, Context + useReducer, responsive UI components.

Tech Stack

  • Backend: ASP.NET Core 8, CoreWCF SOAP, Dapper, PostgreSQL
  • Frontend: React 18, Vite, Tailwind CSS, React Router, Context + useReducer
  • Tooling: xUnit tests, GitHub Pages docs
