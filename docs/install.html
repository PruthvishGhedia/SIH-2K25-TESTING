Installation & Setup

Windows 10/11 with PowerShell 7 recommended. Git Bash optional.

  1. Clone the repository

    git clone https://github.com/your-org/sih-2k25.git
cd sih-2k25

  2. Backend prerequisites

    • .NET 8 SDK
    • PostgreSQL (connection via DATABASE_URL)
    # PowerShell
cd Backend/SIH.ERP.Soap
echo DATABASE_URL="Host=localhost;Database=sih_erp;Username=postgres;Password=yourpassword" > .env
dotnet restore
dotnet build
dotnet run

    Open http://localhost:5000/health to verify health.

  3. Seed data (optional)

    # PowerShell
psql -d sih_erp -f Backend/SIH.ERP.Soap/Seed/seed.sql

  4. Frontend setup

    # PowerShell
cd Frontend
npm install
echo VITE_BACKEND_BASE="http://localhost:5000" > .env.development
npm run dev

    Open http://localhost:5173

  5. Run tests (optional)

    # PowerShell
cd Backend/SIH.ERP.Soap
dotnet test
